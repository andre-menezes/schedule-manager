# Fluxos de UX

## Fluxo principal do usuário

Login → Visualizar agenda do dia → Criar/gerenciar atendimentos → Marcar status.

## Telas do sistema

- Login / Cadastro
- Dashboard (Agenda do dia)
- Pacientes
- Criar/Editar Atendimento
- Detalhe do Atendimento

## Objetivo de cada tela

Dashboard é a tela principal, focada na visualização rápida do dia.

## Estados importantes

Carregando, vazio, erro e sucesso.

## Regras de usabilidade

Poucos cliques, linguagem simples e foco na agenda diária.

- carregamento
- erro
- vazio
- sucesso
